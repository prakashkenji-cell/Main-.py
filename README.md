    <h3>Action games---</h3>
    <p>Action games are games where players must react quickly using hand-eye coordination, timing, and precision. You're usually fighting enemies, dodging attacks, or completing intense missions.</p>
    
    <h4>Main Features</h4>
    <ul>
        <li>Fast-paced gameplay (no waiting, constant movement).</li>
        <li>Combat system (guns, swords, powers).</li>
        <li>Skill-based (reflexes matter more than thinking).</li>
        <li>Levels or missions to complete.</li>
        <li>Boss fights (strong enemies at the end).</li>
    </ul>

    <h3>adventure game---</h3>
    <p>Adventure games focus on narrative (story), solving puzzles, and exploring worlds. Instead of fast combat, you progress by making decisions, interacting with characters, and uncovering secrets.</p>
    
    <h4>Main Features</h4>
    <ul>
        <li>Strong story & characters.</li>
        <li>Exploration (open worlds or detailed environments).</li>
        <li>Puzzles & problem-solving.</li>
        <li>Choices & consequences (your decisions matter).</li>
        <li>Slower pace compared to action games.</li>
    </ul>

    <h3>action+adventure---</h3>
    <p><b>Action-adventure games mix:</b></p>
    <p>⚔️ Action -> fighting, shooting, skills.</p>
    <p>🌍 Adventure -> story, exploration, missions.</p>
    <p>👉 So you <b>fight enemies AND explore a world with a story.</b></p>
    
    <h4>Main Features</h4>
    <ul>
        <li>Open world or semi-open maps.</li>
        <li>Story-based missions.</li>
        <li>Combat (guns / swords / powers).</li>
        <li>Side quests & secrets.</li>
        <li>Character progression (skills, upgrades).</li>
    </ul>
